# Library API Module Documentation

## Base Path
All endpoints in this module are prefixed with `/v1`.

---

## 1. Tree Categories
**Controller**: `TreeCategoriesController`  
**Path**: `/v1/tree-categories`

> [!NOTE]
> Currently, this controller is defined but has no active routes. It is intended for public-facing read operations on tree categories. Administrative operations are handled in the `admin` module.
